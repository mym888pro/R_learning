# R_learning
用python的Q-learnning训练机器人
程序说明
这个程序实现了一个简单的机器人学习系统，包含以下组件：

GridWorld环境：一个5x5的网格世界，包含：

机器人起始位置(0, 0)

目标位置(4, 4)

障碍物位置(1,1), (2,2), (3,3)

Q-learning算法：

使用Q表存储状态-动作值

ε-贪婪策略平衡探索与利用

学习率、折扣因子和探索率可调

训练过程：

训练1000个回合

每个回合机器人尝试从起点到达目标

逐渐减少探索率

测试过程：

使用训练好的Q表指导机器人行动

可视化机器人的移动路径

安装 pip install numpy matplotlib

扩展建议

可以改用更先进的算法如深度Q网络(DQN)
